# 小说阅读器部署指南

本项目是一个纯静态网站，可以免费部署到多个平台。以下是**最简单**的部署方法：

> **⚠️ 重要提示**：如果主要读者在国内，推荐使用 **Gitee Pages** 或 **腾讯云静态网站托管**，访问速度更快且稳定。

## 🇨🇳 方法一：Gitee Pages（国内推荐，访问速度快）

**推荐指数：⭐⭐⭐⭐⭐（国内用户）**

### 步骤：
1. 访问 https://gitee.com
2. 注册/登录账号（支持GitHub账号登录）
3. 点击右上角 `+` → `新建仓库`
4. 填写仓库信息：
   - **仓库名称**：`novel-reader`（或自定义）
   - **是否开源**：选择 `公开`
   - **不要**勾选 "使用Readme文件初始化这个仓库"
5. 点击 `创建`
6. 上传项目文件：
   - 方法A：使用Git命令（推荐）
     ```bash
     git remote add gitee https://gitee.com/你的用户名/novel-reader.git
     git push -u gitee main
     ```
   - 方法B：在网页上直接上传文件
7. 进入仓库 → `服务` → `Gitee Pages`
8. 点击 `启动`，选择分支 `main`，目录 `/`
9. 等待部署完成，获得网址（如：`https://你的用户名.gitee.io/novel-reader`）

**优点：**
- ✅ 国内访问速度快
- ✅ 完全免费
- ✅ 自动HTTPS
- ✅ 支持自定义域名
- ✅ 自动部署（推送代码后自动更新）

**缺点：**
- 需要手动更新（每次推送后需要手动点击"更新"按钮）

---

## 🚀 方法二：Netlify Drop（最简单，无需注册）

**推荐指数：⭐⭐⭐⭐⭐（国外用户）**

### 步骤：
1. 访问 https://app.netlify.com/drop
2. 直接将整个项目文件夹拖拽到页面上
3. 等待几秒钟，自动生成一个网址（如：`https://random-name-123.netlify.app`）
4. 完成！可以直接分享给别人

**优点：**
- 无需注册账号
- 拖拽即用
- 自动HTTPS
- 全球CDN加速

**注意：** 国内可能无法访问

---

## ☁️ 方法三：腾讯云静态网站托管（国内推荐，免费额度）

**推荐指数：⭐⭐⭐⭐⭐（国内用户）**

### 步骤：
1. 访问 https://cloud.tencent.com
2. 注册/登录账号（新用户有免费额度）
3. 进入 **云开发 CloudBase** 控制台
4. 创建环境 → 选择 **静态网站托管**
5. 上传项目文件或连接Git仓库
6. 获得网址（如：`https://你的环境名.cloudbaseapp.com`）

**优点：**
- ✅ 国内访问速度快
- ✅ 免费额度充足（个人项目足够）
- ✅ 自动HTTPS
- ✅ 支持自定义域名
- ✅ 自动部署

**免费额度：**
- 存储：5GB/月
- 流量：5GB/月
- 对于静态网站通常足够

## 🌐 方法四：Vercel（国外推荐，功能强大）

**推荐指数：⭐⭐⭐⭐（国外用户）**

### 步骤：
1. 访问 https://vercel.com
2. 使用 GitHub 账号登录（如果没有，先注册）
3. 点击 "Add New Project"
4. 将项目文件夹拖拽到页面，或连接 GitHub 仓库
5. 点击 "Deploy"，等待完成
6. 获得网址（如：`https://your-project.vercel.app`）

**优点：**
- 免费且稳定
- 自动HTTPS
- 全球CDN
- 支持自定义域名

**注意：** 国内可能无法访问

---

## 📦 方法五：GitHub Pages（适合有GitHub账号）

**推荐指数：⭐⭐⭐**

### 步骤：
1. 在 GitHub 创建新仓库（如：`novel-reader`）
2. 将项目文件上传到仓库
3. 进入仓库 Settings → Pages
4. 选择 Source 为 `main` 分支，`/` 目录
5. 保存后，访问 `https://你的用户名.github.io/novel-reader`

**优点：**
- 完全免费
- 稳定可靠
- 可以绑定自定义域名

**注意：** 国内访问可能较慢或不稳定

---

## ⚡ 方法六：Cloudflare Pages（速度快）

**推荐指数：⭐⭐⭐（国外用户）**

1. 访问 https://pages.cloudflare.com
2. 使用邮箱注册/登录

4. 上传项目文件夹
5. 等待部署完成

**优点：**
- 全球最快的CDN
- 免费且无限制
- 自动HTTPS

**注意：** 国内访问可能较慢

---

## 🇨🇳 方法五：Gitee Pages（国内访问快）

**推荐指数：⭐⭐⭐**

### 步骤：
1. 访问 https://gitee.com
2. 注册账号并创建仓库
3. 上传项目文件
4. 进入仓库 → 服务 → Gitee Pages
5. 开启后获得网址

**优点：**

## 💡 小贴士

1. **自定义域名**：大部分平台都支持绑定自己的域名（如：`novel.yourdomain.com`）
2. **自动更新**：如果连接了Git仓库，修改后会自动重新部署
3. **备份**：建议将项目上传到GitHub/Gitee作为备份

## ❓ 常见问题

**Q: 部署后字体加载慢？**  
A: 字体是从CDN加载的，首次访问可能稍慢，后续会缓存。

**Q: 可以修改网址吗？**  
A: 大部分平台都支持自定义项目名称，生成更友好的网址。

**Q: 部署后如何更新内容？**  
A: 修改本地文件后，重新上传/推送即可自动更新。
