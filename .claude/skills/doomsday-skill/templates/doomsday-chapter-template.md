## 详细说明

### 一、标题区规范

#### 1. Episode 编号格式
```
格式: Episode XX: [集标题]
其中 XX 为两位数编号: 01, 02, 03...60

示例:
# Episode 01: 重生归来
# Episode 15: 天灾降临
# Episode 32: 复仇之夜
# Episode 60: 新世界秩序
```

#### 2. 集标题要求
- **字数**: 4-8个字
- **要素**: 包含核心冲突或爽点关键词
- **风格**: 有冲击力,能吸引读者

**优秀标题示例**:
- ✅ "重生归来" (简洁有力)
- ✅ "疯狂囤货" (直指爽点)
- ✅ "张浩之死" (制造悬念)
- ✅ "冰封末日" (画面感强)

**避免的标题**:
- ❌ "第一天" (太平淡)
- ❌ "林雪瑶开始准备" (太啰嗦)
- ❌ "一些事情发生了" (太模糊)

#### 3. 简介规范
```
格式: **简介**: [内容]
字数: 20-30字
要求: 用一句话精准概括本集核心剧情

示例:
**简介**: 林雪瑶重生回到天灾前七天,决定开始疯狂囤货。
**简介**: 张浩带人围堵避难所,却被林雪瑶的监控系统羞辱。
**简介**: 天灾正式爆发,主角的充足准备让邻居们眼红。
```

#### 4. 关键场景标注
```
格式: **关键场景**: [场景1] | [场景2] | [场景3]
要求: 用3-5个词标注本集主要场景

示例:
**关键场景**: 主角卧室 | 大型超市 | 仓库
**关键场景**: 避难所门口 | 监控室 | 天台
**关键场景**: 冰封街道 | 地下车库 | 对峙现场
```

#### 5. 爽点类型标注
```
格式: **本集爽点**: [类型]

爽点类型分类:
- 囤货爽点: 展现主角物资充裕
- 复仇爽点: 报前世之仇
- 打脸爽点: 打脸嘲讽者
- 装逼爽点: 展现隐藏实力
- 碾压爽点: 实力碾压对手
- 智斗爽点: 智商碾压
- 预知爽点: 利用重生优势

示例:
**本集爽点**: 囤货爽点 - 主角扫空超市,旁人震惊
**本集爽点**: 复仇爽点 - 张浩被拒之门外,前世之仇初显
**本集爽点**: 打脸爽点 - 李娜嘲讽主角,结果被啪啪打脸
```

#### 6. 出场人物标注
```
格式: **出场人物**: [人物A] [人物B] [人物C]

示例:
**出场人物**: 林雪瑶 张浩 超市店员
**出场人物**: 林雪瑶 李娜 王刚
**出场人物**: 林雪瑶(仅独角戏)
```

### 二、正文四段结构详解

#### 【起】场景切入 (100-150字)

**功能**: 快速建立本集场景和焦点

**标准结构**:
```
[1句] 时间/地点交代
[2-3句] 场景视觉描述
[2-3句] 引出本集矛盾/目标
[总计] 6-8句,100-150字
```

**写作要点**:
- ✅ 第一句必须交代时空信息
- ✅ 场景描述要有画面感和氛围
- ✅ 快速引出本集核心矛盾或目标
- ❌ 不要过度铺垫
- ❌ 不要废话寒暄

**A级示例**:
```
天灾爆发前第七天。
林雪瑶睁开眼,看到熟悉的天花板。
手机屏幕上显示: 2024年1月10日,早上六点。

她猛地坐起来。
重生了!
真的重生了!

前世的记忆涌上心头。
冰封的世界,绝望的人群,还有那些害死她的人。
这一次,一切都会不同。
```

**格式标准**:
- 句长: 8-15字
- 总句数: 6-8句
- 节奏: 前慢后快
- 情绪: 由平静到激动/紧张

#### 【承】冲突展开 (200-300字)

**功能**: 推进剧情,展开矛盾

**标准结构**:
```
[5-8句] 冲突初步展开
[5-8句] 人物对话/互动
[3-5句] 矛盾进一步激化
[总计] 15-20句,200-300字
```

**写作要点**:
- ✅ 通过对话和行动推进剧情
- ✅ 冲突要逐步升级
- ✅ 对话要简洁有力,包含信息或冲突
- ✅ 融入人物性格和关系
- ❌ 避免平铺直叙
- ❌ 避免无效对话

**A级示例**:
```
林雪瑶立刻起床。
距离天灾还有七天。
必须抓紧时间。

她打开手机银行。
账户余额: 50万。
前世攒了一年的积蓄,这次要全部用来囤货。

第一站: 大型超市。

林雪瑶推着三辆购物车进入超市。
收银员愣住了。
"三辆?"
"不够,再给我五辆。"

她直奔方便面货架。
整整十箱,全部扫进购物车。
矿泉水,二十箱。
罐头,三十箱。
大米,面粉,食用油......

旁边的顾客侧目。
"这是要开店吗?"
林雪瑶不理会。
她知道,七天后,这些人会后悔今天没多买点。
```

**格式标准**:
- 句长: 10-15字
- 对话占比: 30-40%
- 动作描写: 具体可见
- 节奏: 逐渐加快

#### 【转】爽点高潮 (150-200字)

**功能**: 呈现本集核心爽点

**标准结构**:
```
[3-5句] 主角开始采取行动
[5-8句] 爽点核心呈现
[2-3句] 反派/他人的震惊反应
[总计] 10-15句,150-200字
```

**写作要点**:
- ✅ 爽点呈现要有冲击力
- ✅ 用对比制造反差感
- ✅ 具体数字和细节增强说服力
- ✅ 反派/旁观者的反应烘托爽点
- ❌ 不要点到为止
- ❌ 不要模糊描述

**囤货爽点示例**:
```
收银台前。
八辆购物车堆成小山。
收银员开始扫码。

"一共......"她看着屏幕,瞪大了眼睛。
"十二万三千八百元?"

林雪瑶刷卡。
毫不犹豫。

装车花了三十分钟。
整整一辆小货车,堆得满满当当。

超市经理亲自送到门口。
"女士,明天还来吗?"
"当然。"林雪瑶发动车,"还有六天。"

经理目瞪口呆。
```

**复仇爽点示例**:
```
张浩站在门外。
"林雪瑶!你不能见死不救!"

林雪瑶打开监控。
屏幕上,张浩脸色苍白,嘴唇发紫。
旁边还有四个人,都是前世害死她的。

"救?"
她的声音通过监控传出去。
"你还记得前世,你是怎么把我推进尸群的吗?"

张浩脸色大变。
"你...你在说什么?"

"装傻没用。"
林雪瑶关掉监控。
"这一世,轮到你绝望了。"
```

**格式标准**:
- 句长: 8-12字 (爽点部分用短句增强冲击力)
- 对比强烈: 主角优势 vs 他人窘境
- 数字具体: 用具体数据展现差距
- 反应真实: 旁观者的震惊要合理

#### 【钩】悬念收尾 (50-100字)

**功能**: 制造悬念,引导下集

**标准结构**:
```
[2-3句] 本集事件小结/收尾
[2-3句] 引出新的悬念/威胁
[总计] 4-6句,50-100字
```

**写作要点**:
- ✅ 给本集一个小结局
- ✅ 同时引出新的问题或威胁
- ✅ 制造读者期待感
- ✅ 可以用突然出现的人/事/物制造悬念
- ❌ 不要虎头蛇尾
- ❌ 不要强行制造悬念

**悬念类型**:

**类型1: 新威胁出现**
```
林雪瑶回到家。
物资已经安全存入空间。
第一天,完成。

忽然,门铃响了。
她透过猫眼往外看——
是李娜。

那个前世第一个背叛她的女人。
```

**类型2: 情节转折**
```
张浩带着人离开了。
林雪瑶松了口气。

但她不知道,
张浩已经打电话给了警察。
"警官,我怀疑有人在囤积居奇......"
```

**类型3: 时间推进**
```
第一天,完成。
林雪瑶看着满满的空间。
微微一笑。

还有六天。
她还要买更多。

手机震动。
一条新闻推送: "气象局发布异常天气预警......"
```

**类型4: 人物登场**
```
林雪瑶关上门。
今天的收获不错。

窗外,一个人影站在对面楼顶。
望远镜对准她家。
他放下望远镜,拿起手机:
"目标确认。她果然重生了。"
```

**格式标准**:
- 句长: 6-10字 (用短句增强悬念感)
- 节奏: 突然加速
- 信息: 不要完全透露,留白
- 情绪: 紧张、好奇、期待

### 三、分镜提示区

**功能**: 辅助理解如何将文字转化为漫画分镜

**格式**:
```

- 画面2: [描述]
- 画面3: [描述]

```

**示例**:
```
## 分镜提示

**关键画面**:
- 画面1: 特写手机屏幕显示日期,背景是林雪瑶震惊的脸

**情绪基调**: 前期紧张激动,中期爽快淋漓,结尾悬疑紧张
```

### 五、常见错误对照

#### 错误1: 标题区信息不全
❌ **错误**:
```
# Episode 01: 重生归来

林雪瑶重生了......
```

✅ **正确**:
```
# Episode 01: 重生归来

**简介**: 林雪瑶重生回到天灾前七天,决定开始疯狂囤货。

**关键场景**: 主角卧室 | 大型超市 | 仓库

**本集爽点**: 囤货爽点 - 主角扫空超市,旁人震惊

**出场人物**: 林雪瑶 超市店员 路人

---

## 正文

### 【起】场景切入
......
```

#### 错误2: 四段结构不清晰
❌ **错误**:
```
林雪瑶重生了,她开始买东西,买了很多,然后回家了......
(没有明确的起承转钩分段)
```

✅ **正确**:
```
### 【起】场景切入
天灾爆发前第七天。
林雪瑶睁开眼......

### 【承】冲突展开
她立刻起床......

### 【转】爽点高潮
收银台前......

### 【钩】悬念收尾
林雪瑶回到家......
```

#### 错误3: 字数不达标
❌ **错误**: 300字 (太少)
❌ **错误**: 1200字 (太多)
✅ **正确**: 500-800字

#### 错误4: 句子太长
❌ **错误**:
```
林雪瑶看着眼前堆积如山的物资,心中涌起一股难以言喻的满足感,因为她知道这些东西在天灾来临之后会变得无比珍贵。
```

✅ **正确**:
```
林雪瑶看着满屋物资。
嘴角上扬。
这些东西,在天灾后价值千金。
```

#### 错误5: 缺少爽点
❌ **错误**: 全是铺垫,没有明确爽点
✅ **正确**: 每集至少1个清晰的爽点呈现

#### 错误6: 悬念缺失
❌ **错误**:
```
林雪瑶买完东西回家了。今天过得很充实。
(完全没有悬念)
```

✅ **正确**:
```
林雪瑶买完东西回家了。
今天过得很充实。

但她不知道,有人在暗中盯着她。
一双眼睛,在黑暗中闪烁。
```
