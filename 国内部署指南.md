# 国内部署指南（推荐）

由于 Vercel、Netlify 等国外服务在国内访问可能受限，这里提供**国内可正常访问**的免费部署方案。

## 🏆 方案一：Gitee Pages（最推荐）

### 为什么选择 Gitee Pages？
- ✅ **国内访问速度快** - 服务器在国内
- ✅ **完全免费** - 无使用限制
- ✅ **自动HTTPS** - 安全可靠
- ✅ **支持自定义域名** - 可以绑定自己的域名
- ✅ **操作简单** - 类似 GitHub Pages

### 详细步骤：

#### 1. 注册 Gitee 账号
- 访问：https://gitee.com
- 使用邮箱或手机号注册（也支持 GitHub 账号登录）

#### 2. 创建仓库
1. 点击右上角 `+` → `新建仓库`
2. 填写信息：
   - **仓库名称**：`novel-reader`（或你喜欢的名字）
   - **仓库介绍**：`末世黎明 - 沉浸式小说阅读器`
   - **是否开源**：选择 `公开`
   - **不要**勾选 "使用Readme文件初始化这个仓库"
3. 点击 `创建`

#### 3. 上传代码
**方法A：使用Git命令（推荐）**
```bash
# 添加 Gitee 远程仓库
git remote add gitee https://gitee.com/你的用户名/novel-reader.git

# 推送到 Gitee
git push -u gitee main
```

**方法B：网页上传**
1. 在仓库页面点击 `上传文件`
2. 拖拽项目文件夹中的所有文件
3. 填写提交信息，点击 `提交`

#### 4. 开启 Gitee Pages
1. 进入仓库页面
2. 点击 `服务` → `Gitee Pages`
3. 点击 `启动` 按钮
4. 选择：
   - **分支**：`main`
   - **目录**：`/`（根目录）
5. 点击 `启动`，等待部署完成
6. 获得网址：`https://你的用户名.gitee.io/novel-reader`

#### 5. 自动更新（可选）
每次推送代码后：
1. 进入仓库 → `服务` → `Gitee Pages`
2. 点击 `更新` 按钮
3. 等待重新部署完成

**注意**：Gitee Pages 需要手动点击"更新"，不会自动部署。

## 🏆 方案二：腾讯云静态网站托管

### 为什么选择腾讯云？
- ✅ **国内访问速度快**
- ✅ **免费额度充足**（个人项目足够）
- ✅ **专业稳定**
- ✅ **自动部署**（连接Git后）

### 详细步骤：

#### 1. 注册腾讯云账号
- 访问：https://cloud.tencent.com
- 新用户有免费额度

#### 2. 开通云开发
1. 进入控制台，搜索 "云开发 CloudBase"
2. 创建环境（选择按量计费，有免费额度）
3. 等待环境创建完成

#### 3. 开启静态网站托管
1. 在环境管理页面，点击 `静态网站托管`
2. 点击 `开通`
3. 选择 `基础版`（免费）

#### 4. 上传文件
**方法A：网页上传**
1. 点击 `上传文件`
2. 选择项目文件夹中的所有文件
3. 上传完成

**方法B：CLI工具（推荐）**
```bash
# 安装 CloudBase CLI
npm install -g @cloudbase/cli

# 登录
tcb login

# 部署
tcb hosting deploy . -e 你的环境ID
```

#### 5. 访问网站
- 获得默认域名：`https://你的环境ID.cloudbaseapp.com`
- 可以绑定自定义域名

**免费额度：**
- 存储：5GB/月
- 流量：5GB/月
- 对于静态网站通常足够使用

## 🏆 方案三：阿里云 OSS + CDN

### 为什么选择阿里云？
- ✅ **国内访问速度快**
- ✅ **CDN加速**
- ✅ **专业稳定**

### 详细步骤：

#### 1. 注册阿里云账号
- 访问：https://www.aliyun.com
- 新用户有免费额度

#### 2. 开通 OSS
1. 进入 OSS 控制台
2. 创建 Bucket（存储桶）
3. 设置：
   - **读写权限**：公共读
   - **地域**：选择离你近的（如：华东1）

#### 3. 上传文件
1. 进入 Bucket
2. 点击 `上传文件`
3. 选择项目文件夹中的所有文件
4. 上传完成

#### 4. 开启静态网站托管
1. 在 Bucket 设置中，找到 `静态网站托管`
2. 点击 `设置`
3. 设置：
   - **默认首页**：`index.html`
   - **默认404页**：`index.html`（可选）
4. 保存

#### 5. 访问网站
- 获得 OSS 访问地址：`https://你的Bucket名.oss-region.aliyuncs.com`
- 可以绑定自定义域名

**免费额度：**
- 存储：5GB/月
- 流量：5GB/月
- 请求：100万次/月

## ❓ 常见问题

**Q: Gitee Pages 需要手动更新，有没有自动更新的方法？**
A: 可以使用 GitHub Actions 或 Gitee Go 实现自动部署，但需要配置。

**Q: 免费额度用完了怎么办？**
A: 对于纯静态网站，5GB/月的免费额度通常足够。如果超出，可以考虑：
- 压缩文件
- 使用 CDN 缓存
- 升级到付费套餐（价格通常很低）

**Q: 可以同时部署到多个平台吗？**
A: 可以！建议同时部署到 Gitee Pages 和腾讯云，确保可用性。
