# 职场逆袭漫剧创作技能包

## 技能包说明
这是一个专业的职场逆袭题材漫剧创作技能包，包含完整的创作方法论、写作风格定义、角色塑造体系和实战模板。专注于创作"小人物逆袭打脸、职场晋升"类型的60集短剧。

## 技能包结构
```
workplace-revenge/
├── SKILL.md                           # 当前文件，技能包核心配置
├── workplace-revenge-method.md       # 职场逆袭创作方法论
├── workplace-revenge-style.md         # 写作风格定义
├── templates/                         # 文档格式模板
│   ├── workplace-outline-template.md
│   ├── workplace-character-template.md
│   ├── workplace-chapter-index-template.md
│   └── workplace-chapter-template.md
└── examples/                          # 创作示例
    ├── outline-example.md
    ├── character-example.md
    ├── catalog-example.md
    └── chapter-example.md
```

## 核心能力
1. **职场逆袭结构设计**：60集三幕式晋升路径规划
2. **角色塑造体系**：主角-反派-配角-导师的层级设计
3. **爽点分布策略**：打脸爽点+能力觉醒+晋升荣耀的节奏控制
4. **单集叙事结构**：起承转钩的500-800字标准格式
5. **职场真实性**：符合现代职场环境和晋升逻辑
6. **情感线管理**：职场恋情+师徒情+竞争关系的平衡

## 职场逆袭题材特点

### 核心理念
- **现实性**：基于真实职场环境和晋升逻辑
- **爽感密集**：每集1-2个打脸或能力展示爽点
- **成长路径**：从底层到高层的完整晋升体系
- **情感共鸣**：观众对职场不公的情感投射

### 热门设定
1. **行业背景**：互联网、金融、广告、地产、医药等
2. **主角身份**：实习生、管培生、被欺负的助理、被埋没的人才
3. **金手指类型**：重生记忆、专业技能、贵人相助、系统任务
4. **反派设定**：势利眼上司、抢功劳同事、家族企业继承人
5. **晋升路径**：员工→组长→主管→经理→总监→副总裁→总裁

### 爽点体系
1. **能力打脸**：专业技能展示、解决重大问题
2. **身份揭秘**：隐藏背景、真实能力曝光
3. **业绩碾压**：业绩爆表、超越预期
4. **职场正义**：揭露阴谋、讨回公道
5. **晋升荣耀**：升职加薪、获得认可

## 执行流程
1. 主Agent识别当前创作阶段
2. 自动调用对应的方法论、模板、示例文件
3. 基于skill提供的专业知识进行内容创作
4. 严格遵循模板格式输出结果
5. 正文阶段完成后触发质量检查

## Commands

### /status
显示当前创作进度：
- 已完成的阶段（大纲/人物/目录/正文）
- 正文创作进度（已完成X/60集）
- 下一步操作建议

### /outline
重新生成或修改故事大纲（会询问是否覆盖现有大纲）

### /character
创建或修改人物小传（需要先完成大纲）

### /catalog
生成或修改分集目录（需要先完成大纲和人物）

### /write
开始或继续正文创作（需要先完成大纲、人物、目录）

### /check
手动触发质量检查（检查最近完成的5集内容）

### /help
显示完整指令列表和使用说明

## 核心原则

1. **现实性优先**：所有剧情必须符合现代职场逻辑
2. **模板驱动**：所有输出严格遵循 templates/ 定义的格式
3. **质量保障**：正文创作每5集后自动触发质量检查
4. **爽点密集**：确保每集都有打脸或能力展示内容
5. **晋升路径**：严格按照职级晋升，避免跳跃式发展

## 注意事项

- 输入质量决定输出质量：用户提供的创意越详细，生成内容越贴合预期
- 职场真实性：避免超现实设定，保持职场环境的可信度
- 晋升逻辑：主角的成长必须有明确的能力支撑
- 人物塑造：避免脸谱化，每个角色都要有独特性格

## 备注
本技能包专注于职场逆袭题材，通过系统化的创作方法论和模板体系，帮助创作高质量的职场逆袭漫剧。重点关注现实性、爽感密度和观众共鸣。
