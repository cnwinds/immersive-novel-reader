# 《末世黎明》世界观设定文档

---

## 一、末世起源

### **病毒真相（逐步揭示）**
1. **表层认知**：2025年，一种名为"H7N9-Z"的流感病毒变异，全球大流行
2. **第二层真相**：病毒中含有远古病毒样本，来自南极冰层下的未知文明遗迹
3. **第三层真相**：这是外星文明留下的"筛选机制"，目的不明
4. **最终真相**：地球每5000年一次的"进化清洗"，只有适者才能进入下一轮文明

### **双阶段爆发**
- **第一阶段（感染期）**：2025年8月-10月
  - 类似普通流感，90%人口感染
  - 感染者会获得轻微的身体强化（不明显）
  - 为第二阶段做准备

- **第二阶段（突变期）**：2025年11月1日
  - 全球范围内同时爆发
  - 感染者在3小时内丧尸化
  - 体质特殊者会觉醒异能

## 二、丧尸体系

### **丧尸等级**
1. **普通丧尸（白眸）**
   - 特征：行动迟缓，无智慧
   - 战力：普通人持武器可击杀
   - 晶核：无

2. **迅捷丧尸（灰眸）**
   - 特征：速度极快，有简单本能
   - 战力：需3-5名普通人协作
   - 晶核：白色晶核（概率20%）

3. **力量丧尸（黄眸）**
   - 特征：体型庞大，力量惊人
   - 战力：需重武器或异能者
   - 晶核：白色晶核（概率50%）

4. **变异丧尸（绿眸）**
   - 特征：保留部分智慧，会使用简单工具
   - 战力：需多名一级异能者
   - 晶核：绿色晶核（100%）

5. **领主丧尸（蓝眸）**
   - 特征：可指挥低级丧尸，有领地意识
   - 战力：需二级以上异能者团队
   - 晶核：蓝色晶核（100%）

6. **君王丧尸（紫眸）**
   - 特征：完全智慧，能伪装人类
   - 战力：需三级异能者或军队
   - 晶核：紫色晶核（100%）

7. **未知等级（红眸/金眸）**
   - 仅在传闻中出现

---

## 三、进化体系

### **异能者等级**
1. **觉醒者（Level 1）**：刚觉醒，能简单使用能力
2. **强化者（Level 2）**：能力大幅增强，可持续战斗
3. **掌控者（Level 3）**：能力运用自如，开发新用法
4. **统领者（Level 4）**：能力质变，可影响他人
5. **君王级（Level 5）**：传说中的等级

### **进化途径**
1. **晶核吸收**：最常见，但有瓶颈和风险
2. **生死战斗**：极限状态下突破
3. **神秘遗址**：获得远古传承
4. **特殊物品**：黑晶核等神秘物品

### **异能分类**
- **元素系**：火、水、冰、雷等
- **强化系**：力量、速度、体质强化
- **精神系**：心灵感应、念力、幻术
- **特殊系**：空间、时间、治愈、预知
- **混沌系**：林夜专属，可融合多种能力

---

## 四、神秘遗址

### **遗址类型**
1. **小型遗址**：密室、古墓，少量物资
2. **中型遗址**：地下设施、军事基地，有防御机制
3. **大型遗址**：远古文明遗迹，有守护者
4. **特殊遗址**：移动型、空间型

### **遗址机制**
- **谜题解锁**：需要特定条件（血脉、物品、时间）
- **守护者**：强力丧尸或机关
- **奖励**：晶核、装备、进化药剂、远古技术
- **危险**：陷阱、辐射、精神污染

### **关键遗址**
1. **第一遗址（制药厂地下）**：林夜据点，获得进化装置
2. **第二遗址（南山古墓）**：病毒起源线索
3. **第三遗址（东海海底城）**：最终真相

## 五、世界格局

### **末世前期（1-30天）**
- 全球通讯中断
- 90%人类死亡或丧尸化
- 幸存者各自为战
- 小型幸存者团体出现

### **末世中期（1-6个月）**
- 异能者体系确立
- 大型幸存者基地建立
- 军阀割据，互相征伐
- 神秘遗址被发现

### **末世后期（6个月后）**
- 势力版图基本确定
- 丧尸形成"王国"
- "守望者"等神秘组织浮出水面
- 真相开始揭示

### **主要势力**
1. **新黎明避难所**（林夜势力）：技术+精英路线
2. **钢铁城**：军阀+强权路线
3. **自由联盟**：民主+联邦路线
4. **圣火教**：宗教+狂热路线
5. **守望者组织**：神秘+中立路线

## 六、装备体系

### **武器等级**
1. **普通武器**：现代枪械、冷兵器
2. **强化武器**：掺入晶核粉末的武器，对丧尸伤害加成
3. **异能武器**：与使用者绑定，可成长
4. **远古武器**：遗址中获得，威力巨大

### **防具等级**
1. **普通防具**：防弹衣、防护服
2. **强化防具**：抗腐蚀、抗撕裂
3. **异能防具**：有主动技能
4. **远古防具**：自动修复、能量护盾

### **特殊装备**
- **空间装备**：储物戒指等（极度稀有）
- **探测装备**：丧尸探测器、晶核探测器
- **通讯装备**：短波电台、加密通讯器

## 七、特殊物品

### **晶核体系**
- **白色晶核**：普通丧尸，用于基础进化
- **绿色晶核**：变异丧尸，价值高
- **蓝色晶核**：领主丧尸，稀有
- **紫色晶核**：君王丧尸，极度稀有
- **黑色晶核**：特殊，林夜专属，可融合其他晶核
- **金色晶核**：传说中的存在

### **其他物品**
- **进化药剂**：人工合成晶核能量，有副作用
- **稳定剂**：防止丧尸化
- **远古文献**：记载病毒真相
- **神秘金属**：制作高级装备

## 九、核心概念

### **混沌理论**
- 黑晶核代表着"混沌"，是变数
- 林夜的重生打破了既定命运
- 引发蝴蝶效应，历史开始偏离轨道

### **进化选择**
- 病毒是筛选工具，不是毁灭工具
- 适者生存，不适者被淘汰
- 最终目的是催生出"新人类"

### **文明轮回**
- 地球文明经历过多次轮回
- 每次都有幸存者进入下一轮
- 这次是第七次轮回

## 十、第一季核心谜题

1. **黑晶核的来历**：为什么能让林夜重生？
2. **病毒起源**：是意外还是人为？是自然还是外星？
3. **沈父的下落**：消失的考古学家去了哪里？
4. **钢铁城城主身份**：为何对林夜特别关注？
5. **守望者目的**：他们真的是正义的吗？
6. **远古文明**：上一轮文明的幸存者去了哪里？
7. **最终清洗**：所谓的清洗到底是什么？

## 十一、创作要点

### **风格基调**
- 写实硬核：强调生存艰辛，人性的复杂
- 节奏紧凑：每集都有冲突和推进
- 信息留白：逐渐揭示真相，保持神秘感

### **视觉元素**
- 色调：前期偏暖（回忆），末世后偏冷（现实）
- 对比：繁华都市 vs 废墟，人性光辉 vs 黑暗
- 特效：简洁有力，不夸张

### **叙事技巧**
- 闪回手法：林夜前世的记忆片段
- 多线叙事：主角线 + 配角线 + 反派线
- 悬念设置：每集结尾留钩子
