## 🚀 替代方案：使用腾讯云静态网站托管（推荐）

### 为什么选择腾讯云？
- ✅ **国内访问速度快** - 服务器在国内
- ✅ **免费额度充足** - 5GB存储 + 5GB流量/月（个人项目足够）
- ✅ **自动部署** - 连接 Git 后自动更新
- ✅ **专业稳定** - 腾讯云基础设施
- ✅ **完全免费** - 个人项目通常不会超出免费额度

### 详细部署步骤

#### 第一步：注册腾讯云账号
1. 访问：https://cloud.tencent.com
2. 注册账号（新用户有免费额度）

#### 第二步：开通云开发 CloudBase
1. 登录腾讯云控制台
2. 搜索 **"云开发 CloudBase"**
3. 点击进入，选择 **"新建环境"**
4. 填写信息：
   - **环境名称**：`novel-reader`（或你喜欢的名字）
   - **计费方式**：选择 **"按量计费"**（有免费额度）
   - **地域**：选择离你近的（如：上海）
5. 点击 **"立即开通"**，等待环境创建完成（约1-2分钟）

#### 第三步：开启静态网站托管
1. 在环境管理页面，找到 **"静态网站托管"**
2. 点击 **"开通"** 按钮
3. 选择 **"基础版"**（免费）
4. 等待开通完成

#### 第四步：上传项目文件
**方法A：网页上传（最简单）**
1. 在静态网站托管页面，点击 **"上传文件"**
2. 选择项目文件夹中的所有文件（包括 `index.html`、`css/`、`js/` 等）
3. 上传完成

**方法B：使用 CLI 工具（推荐，支持自动部署）**
```bash
# 1. 安装 CloudBase CLI
npm install -g @cloudbase/cli

# 2. 登录（会打开浏览器进行授权）
tcb login

# 3. 部署项目（在项目根目录执行）
tcb hosting deploy . -e 你的环境ID
```

#### 第五步：访问网站
- 部署完成后，会显示默认域名：`https://你的环境ID.cloudbaseapp.com`
- 可以绑定自定义域名（可选）

### 免费额度说明
- **存储空间**：5GB/月
- **流量**：5GB/月
- **对于静态小说阅读器，通常不会超出免费额度**

### 网页手动更新

2. 删除旧文件，重新上传新文件
3. 等待部署完成

**提示**：使用 CLI 工具可以设置 Git Hook，实现自动部署。

## 🔄 连接 Git 仓库实现自动部署

腾讯云 CloudBase 支持连接 Git 仓库，实现自动部署：

1. 在静态网站托管页面，找到 **"Git 部署"**
2. 选择 **"Gitee"** 或 **"GitHub"**
3. 授权连接你的仓库
4. 配置：
   - **分支**：`main`
   - **构建命令**：留空（静态网站不需要构建）
   - **输出目录**：`/`（根目录）
5. 保存后，每次推送代码会自动部署

这样就不需要手动上传文件了！

## ✨ 自定义域名（可选）

1. 在 Gitee Pages 设置中，找到 **"自定义域名"**
2. 输入你的域名（如：`novel.yourdomain.com`）
3. 在你的域名 DNS 设置中添加 CNAME 记录：
   - **类型**：CNAME
   - **主机记录**：`novel`（或 `@`）
   - **记录值**：`cnwinds.gitee.io`

## 🆘 常见问题

**Q: 免费额度用完了怎么办？**
A: 对于纯静态网站，5GB/月的免费额度通常足够。如果超出，可以考虑：
- 压缩文件（CSS、JS）
- 使用 CDN 缓存
- 升级到付费套餐（价格通常很低）

**Q: 部署失败怎么办？**
A: 检查：
- 确保 `index.html` 在根目录
- 确保所有文件都已上传
- 查看部署日志了解错误原因

**Q: 如何查看部署日志？**
A: 在静态网站托管页面，点击 **"部署日志"** 可以查看详细日志。

**Q: 可以绑定自定义域名吗？**
A: 可以！在静态网站托管设置中，找到 **"自定义域名"**，按照提示配置即可。

**Q: 还有其他替代方案吗？**
