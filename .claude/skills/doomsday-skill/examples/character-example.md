# 人物小传示例

## 主角

### 基本信息
- **姓名**：陈意宁
- **性别**：女
- **年龄**：25岁
- **前世身份**：普通白领，市场部专员
- **今世身份**：重生者，末世幸存者领袖

### 前世经历
**前世性格**：善良、单纯、容易信任他人、优柔寡断、圣母心

**前世遭遇**：
- 极寒天灾爆发后，陈意宁带着囤积的少量物资躲在出租屋
- 第三天，闺蜜林雪和前男友张浩上门求助，她毫不犹豫分享食物和空间手镯
- 第十天，林雪提议大家一起去寻找物资，途中设局将陈意宁推下楼
- 陈意宁腿部骨折，被林雪和张浩抢走空间手镯和所有食物
- 两人将她锁在废弃仓库，留下一句"对不起，但我们也想活"

**死亡情节**（必须详细）：
- **死亡时间**：天灾爆发后第九十天
- **死亡地点**：城南废弃仓库三楼
- **死因**：冻死+饿死（零下八十度，无食物无保暖）
- **害死主角的人**：闺蜜林雪、前男友张浩
- **害死的原因**：抢夺空间手镯和物资，林雪从高中起就喜欢张浩，接近陈意宁只为抢男友，末世成为她除掉情敌的机会
- **死前心情**：绝望、愤怒、不甘——"如果有来世，我绝不会再心软"

**前世教训**：
- 善良要有锋芒，不能对所有人都好
- 看清人性本质，林雪的虚伪和张浩的自私
- 准备永远不嫌早，囤货要够全面
- 信任要有代价，不能轻信任何人

### 重生设定
- **重生时间点**：天灾爆发前三个月（距离零下四十度爆发还有90天）
- **重生地点**：城中区出租屋，早上六点醒来
- **重生触发**：闹钟响起的瞬间，前世记忆涌入脑海，右手空间手镯发烫
- **重生记忆**：保留前世完整记忆，包括天灾时间线、人物真面目、资源分布点

### 今世性格
**性格转变**：
- 前世 → 今世
- [善良] → [有原则的善良，只对值得的人好]
- [懦弱] → [果断决绝，该出手时不犹豫]
- [信任他人] → [保持警惕，用行动而非言语判断人]
- [优柔寡断] → [目标明确，手段狠辣]
- [圣母心] → [冷酷理智，但保留底线]

**行为准则**：
- ? 会做：囤货准备、保护家人、复仇前世仇人、建立可控团队、用资源换忠诚
- ? 不会做：滥用善良、原谅背叛者、心软放过敌人、无限制帮助陌生人、伤害无辜者

**核心特质**：冷静、果断、记仇、重情（对家人）、狠辣（对敌人）

### 金手指
- **类型**：空间异能
- **获得方式**：重生时自动升级（前世10平米→今世100平米）
- **初始能力**：100平米储物空间，时间流速1:10（外界1天=空间10天），可种植
- **能力限制**：无法储存活人，物品需手动放入，种植需种子土壤
- **成长路径**：第40集→200平米+冰冻攻击，第52集→500平米+冰封领域，第58集→1000平米+领域扩大

### 目标与动机
**短期目标**（第1-20集）：
- 囤货：五百袋米、三百袋面、两千罐罐头、充足燃料和药品
- 保护家人：说服父母进城，一起准备过冬
- 初步复仇：打脸房东、同事、前男友，远离林雪和张浩

**中期目标**（第21-45集）：
- 建立安全基地：打造温暖据点，组建核心团队
- 复仇林雪张浩：夺回空间手镯，让他们体验前世加倍的痛苦
- 能力升级：解锁更强大的空间异能

**终极目标**（第46-60集）：
- 清算前世仇人：让林雪和张浩生不如死
- 击败终极威胁：对抗重生者周寒，夺取避难所
- 建立新秩序：在冰封世界成为领导者，保护重要的人

### 成长路径
- **第1-20集状态**：重生者菜鸟，以囤货和避险为主，能力100平米
- **第21-45集状态**：基地领袖，展现领导力，能力升级到200平米，掌握冰冻攻击
- **第46-60集状态**：异能者强者，冰封领域震撼全场，成为区域最强者

### 关系网络
- **家人**：父母，从老家接来，第18集开始同住，关系亲密，父亲务实母亲温柔
- **感情线对象**：沈默，老赵外甥，退伍特种兵，火系异能者，第43集定情
- **忠诚队友**：老赵（退伍军人，50岁），小七（老赵女儿，12岁），团队管家型人物
- **前世仇人**：林雪（闺蜜变仇人），张浩（前男友变仇人）

### 功能定位
- 推动剧情：囤货、复仇、基地建设、终极对决全部主线
- 提供爽点：囤货展示、打脸复仇、能力碾压、智谋布局
- 情感核心：亲情（父母）、爱情（沈默）、复仇（林雪张浩）

## 炮灰反派（3-5位）

### 炮灰反派1：王大妈

**基本信息**：
- **姓名**：王淑芬（王大妈）
- **身份**：房东，60岁，嫌贫爱富
- **年龄**：60岁

**前世作恶**：
- **作恶时间**：天灾爆发后第五天
- **作恶内容**：带着儿子和邻居三家人，用撬棍砸开陈意宁房门，抢走大米面粉和棉被，还打伤陈意宁导致她腿部骨折无法逃生
- **作恶原因**：平时看不起陈意宁这个外地打工妹，末世想起她可能囤了货，贪念作祟
- **对主角的伤害**：物资被抢，腿被打断，为后续被林雪害死埋下伏笔

**今世打脸**：
- **出场集数**：第6集（上门催房租时发现陈意宁在囤货，起贪心）
- **打脸集数**：第12集
- **打脸情节**：陈意宁提前搬家到安全小区，第12集王大妈天灾后带人找来求助，陈意宁当众展示电击棒，冷笑："前世你抢我东西还打断我的腿，今世，门都别想进。"王大妈吓得瘫软在地，被赶走
- **打脸结局**：第25集路过时看到王大妈一家在垃圾堆翻找食物，冻得瑟瑟发抖

**性格特点**：势利眼、贪婪、欺软怕硬、自私、无底线

**功能定位**：提供第一次打脸爽点，展示主角重生后的果断和冷酷

## 前世仇人（1-2位核心反派）

### 前世仇人1：林雪

**基本信息**：
- **姓名**：林雪
- **性别**：女
- **年龄**：25岁
- **前世身份**：陈意宁闺蜜，外企HR
- **今世身份**：伪装者，小团体头目

**前世关系**：
- **与主角的关系**：高中到大学的闺蜜，表面亲密无间
- **关系起点**：高一同班，林雪主动示好，两人成为好友
- **关系转折**：大二时陈意宁和张浩在一起，林雪开始嫉妒（她暗恋张浩多年）

**前世罪行**（必须详细）：
- **罪行时间**：天灾爆发后第十天
- **罪行内容**：
  - 提议一起外出找物资，实则设局推陈意宁下楼
  - 抢走空间手镯（前世陈意宁告诉她手镯的秘密）
  - 搜刮陈意宁所有食物和保暖衣物
  - 将陈意宁锁在废弃仓库，留下一句"对不起，但我更想活"
  - 和张浩扬长而去，用抢来的物资建立小团体
- **罪行动机**：
  - 从高中就喜欢张浩，接近陈意宁是为了接近他
  - 陈意宁和张浩在一起后心生嫉妒
  - 末世是除掉情敌的绝佳机会，既能得到张浩，又能抢到空间
- **罪行后果**：陈意宁腿部骨折，被锁仓库，第九十天冻死饿死

**性格特点**：
- **表面**：温柔、善解人意、关心朋友、善良大方
- **实际**：虚伪、嫉妒心重、工于心计、自私冷血、为达目的不择手段
- **核心特质**：伪装、嫉妒、狠毒、贪婪、冷血

**今世发展**：
- **初次出场**：第5集（发消息关心陈意宁，约吃饭）
- **主角识破**：第8集，陈意宁看穿林雪接近自己是为了张浩，开始冷淡
- **首次交锋**：第17集，林雪借口缺钱想借五万，陈意宁拒绝并冷嘲"你不是要和张浩在一起吗？找他借"
- **中期对抗**：第33-45集，林雪天灾后建立小团体，多次试图渗透陈意宁基地，第35集资源争夺战，第38集设陷阱被识破，第42集手下被反杀三人
- **复仇高潮**：第48集，陈意宁设局引林雪团队抢发电机，在冰雪中伏击
- **最终结局**：第52集，张浩为救林雪冻死，林雪跪地求饶，陈意宁夺回空间手镯，冷笑："前世你锁我在仓库，今世我让你体验零下九十度的绝望。"将林雪驱逐出基地，生死不明

**复仇规划**：
- **第一阶段**（第5-20集）：识破林雪真面目，保持距离，拒绝一切示好
- **第二阶段**（第21-45集）：天灾后林雪暴露本性，两个团体对抗，陈意宁逐步削弱对方
- **第三阶段**（第46-52集）：设局伏击，夺回空间手镯，看着林雪崩溃求饶
- **终极清算**（第52集）：驱逐林雪，让她在零下九十度中自生自灭，比前世的仓库更残酷

**能力与资源**：
- **个人能力**：善于伪装，擅长笼络人心，有一定管理能力
- **掌握资源**：前期靠抢劫积累少量物资，中期建立二十人小团体，后期资源耗尽
- **威胁等级**：6/10（前期威胁不大，中期有一定实力，后期走投无路最危险）

**功能定位**：
- 推动剧情：复仇主线核心人物，贯穿全剧的主要冲突
- 提供爽点：多次打脸场景，第17集拒绝借钱，第33集嘲讽，第42集反杀，第52集终极复仇
- 情感冲击：背叛的愤怒、复仇的快感、清算的畅快
