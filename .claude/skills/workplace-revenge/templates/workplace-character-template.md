- 管理能力差，无法带领团队

- 品格有问题，失去团队信任
- 心胸狭窄，无法容纳人才

## 创作检查清单

完成人物小传前，确认以下要素：
- [ ] 主角是否有清晰的成长路径？
- [ ] 反派是否有合理的失败原因？
- [ ] 导师是否有存在价值和作用？
- [ ] 盟友是否有功能性定位？
- [ ] 角色关系是否复杂合理？
- [ ] 人物性格是否符合职场现实？
- [ ] 职业发展是否符合逻辑？
- [ ] 情感线是否自然发展？
- [ ] 反派数量是否适中（3-5个）？
- [ ] 是否避免脸谱化人物？

## 备注
- 职场逆袭题材的成功关键在于人物的真实性和成长性
- 主角的成长必须有明确的能力支撑和业绩证明
- 反派的失败要体现出"善有善报，恶有恶报"的价值观
- 导师和盟友的存在要让主角的成长更加合理可信
- 每个角色都要有存在的必要性，避免多余人物
