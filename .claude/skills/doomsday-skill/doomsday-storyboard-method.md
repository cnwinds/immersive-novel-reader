# 末世重生漫剧分镜脚本创作方法论

## 分镜脚本概述

分镜脚本是将文字剧情转化为视觉化画面的桥梁。对于末世重生题材的漫剧，需要通过精确的镜头语言、画面构图和节奏控制，完美呈现囤货爽感和复仇打脸爽感。

## 核心规则

### 1. 严格时长控制
- **强制规则**：每个单独镜头的内容时长必须 ≤ 10秒
- **强制拆分原则**：若故事内容超过10秒，必须拆分为多个独立镜头，绝不允许单镜头超时
- **动作序列时长**：每个动作序列必须在5-10秒内完成

### 2. 视角分类与适配

根据漫剧类型，分为三种视角：

**A. 自拍视角**
- 适用场景：主角主观视角、日常vlog风格
- 景别选择："自拍角度"、"手机自拍视角"、"近距离自拍"
- **禁止元素**：不能出现"手持自拍杆"、"拿着手机"等会破坏视角代入感的元素

**B. 第一人称视角**
- 适用场景：主角沉浸式体验、紧张刺激场面
- 景别选择："第一人称视角"、"主观镜头"
- **禁止元素**：不应出现主角的正面全貌

**C. 第三人称视角**
- 适用场景：全面展现剧情、人物关系和场景布局
- 景别选择："远景"、"中景"、"近景"、"特写"等传统分类
- **特点**：可以有观察者视角的完整画面

### 3. 角色一致性强制要求

**每个分镜的提示词必须完整重复主角的外观描述**：
- 年龄
- 性别
- 服装
- 发型
- 其他面部特征

**目的**：确保跨镜头的角色视觉一致性，防止AI生图产生角色形象漂移。

### 4. 文生图提示词要求

#### 格式结构
```
{景别选择}，{角色外观}，{主体描述}，{环境场景}，{情绪氛围}，{视觉风格}，{光影效果}
```

#### 关键规则

**A. 物理一致性**
- 视角、光照、比例不能违背常识
- 第一帧必须是静态的瞬间画面，而非"边A边B"的持续动作

**B. 露脸策略明确**
- **后续需要露脸** → 第一帧就要露脸
- **后续无需露脸** → 只能给无脸特写或背影

**C. 物品与空间一致性**
- 跨镜头的道具位置、朝向与房间布局保持完全一致
- 防止"生图幻觉"导致的物品漂移

**D. 场景切换规则**
- **每组镜头换场景**：相邻镜头组必须在不同空间开场
- 示例：客厅 → 书房 → 玄关
- 目的：避免布局冲突和视觉混乱

**E. 适度语义模糊**
- 当过细的描述容易导致AI理解错误时，使用更宽泛的用词
- 示例："按掉手机" 胜过 "按掉手机闹铃"（降低细节要求，提高成功率）

### 5. 图生视频提示词要求

#### 格式结构
```
{镜头描述}，{镜头运动}，{主体行为}，{背景变化}，{环境变化}
```

#### 关键规则

**A. 严格时长控制**
- 每段动作描述必须能在5-10秒内完成
- 核心动作、台词、氛围、要求全部写在一段内，避免分散

**B. 动作链清晰**
- 使用明确的串联词："→"、"立刻"、"紧接着"
- 确保动作之间的因果逻辑明确

**C. 关键要求突出**
- 在提示词结尾用一句话强调稳定性要点
- 示例："保持角色面部清晰、光线稳定"

**D. 首帧承接规则**
- 图生视频的起始帧必须完全沿用对应文生图的静态首帧
- 确保人物与道具完全一致，无缝衔接

## 分镜制作流程

### 【步骤1】故事理解

**执行时机**：收到用户请求后

**输入要求**：
1. 故事内容（可接受两种方式）
   - 方式一：直接输入故事梗概（一句话到几段话都可以）
   - 方式二：上传已写好的剧本文件（建议不要一次性上传太多集）

2. 用户必须明确回答的三个问题：
   - **Q1**：目标时长？（30秒短视频 / 3分钟短片 / 更长）
   - **Q2**：希望什么风格？（电影感 / 动画风 / 写实 / 卡通等）
   - **Q3**：视频是什么视角？（自拍视角 / 第一人称视角 / 第三人称视角）

**工作流程**：
1. 确认收到用户提供的全部信息
2. 分析故事总时长
3. 按照时长限制初步估算需要的镜头数量
4. 进入【步骤2】分镜制作

### 【步骤2】分镜制作

**核心任务**：严格按照规则生成分镜脚本

**执行流程**：
1. **分析总时长**：明确整个视频的时长限制
2. **强制拆分**：根据"每个镜头≤10秒"的原则，提前规划镜头数量
3. **逐个生成**：按故事发展顺序，为每个镜头生成以下内容：
   - 镜头编号
   - 时长（5-10秒之间）
   - 场景描述（动作、对白、情绪）
   - 文生图提示词（遵循格式和规则）
   - 图生视频提示词（遵循格式和规则）
   - 备注（特殊要求、注意事项）

4. **质量检查**：全部分镜生成完成后，执行强制时长检查
   - 逐一核对每个镜头的时长
   - 如发现任何镜头>10秒，立即拆分
   - 循环检查直到所有镜头都≤10秒

**输出格式**：
- 使用Markdown表格形式
- 表头：镜头 | 时长 | 场景描述 | 文生图提示词 | 图生视频提示词 | 备注
- 每个字段必须填写完整

## 提示词工程技巧

### 文生图高级技巧

1. **视角关键词库**
   - 自拍视角："手机前置摄像头视角"、"近距离自拍"、"手持设备视角"、"45度角自拍"
   - 第一人称："POV"、"主观镜头"、"通过主角眼睛看"、"沉浸式视角"
   - 第三人称："全景"、"中景"、"近景"、"特写"、"广角"、"鸟瞰视角"

2. **风格关键词库**
   - 电影感："电影光照"、"胶片质感"、"35mm电影镜头"、"景深效果"、"色彩分级"
   - 动画风："动漫风格"、"二次元"、"赛璐珞着色"、"线条清晰"、"平面化阴影"
   - 写实："真实质感"、"超精细细节"、"8K画质"、"照片级真实"、"体积光照"
   - 卡通："卡通渲染"、"夸张比例"、"明亮色彩"、"简化阴影"、"漫画风格"

3. **情绪氛围关键词库**
   - 爽感："张力十足"、"戏剧性光照"、"强烈对比"、"动感十足"
   - 末世："阴沉天空"、"废墟质感"、"冷色调"、"压抑氛围"
   - 重生："温暖色调"、"希望之光"、"柔和边缘光"、"新生感"
   - 囤货："丰富细节"、"质感突出"、"金属反光"、"数量感"
   - 复仇："锐利光线"、"冷峻表情"、"低角度拍摄"、"强势构图"

4. **光影效果关键词库**
   - 室内："柔光箱光照"、"窗光"、"台灯照明"、"阴影柔和"
   - 室外："黄金时刻阳光"、"阴天漫反射"、"强烈逆光"、"体积光"
   - 特殊："伦勃朗光照"、"轮廓光"、"单光源"、"丁达尔效应"

### 图生视频高级技巧

1. **镜头运动关键词**
   - 推轨："镜头推进"、"缓慢推近"、"向前移动"
   - 拉轨："镜头后撤"、"逐渐拉远"、"向后移动"
   - 摇摄："镜头平移"、"左右晃动"、"360度旋转"
   - 固定："镜头静止"、"稳定画面"、"无运动"
   - 手持："轻微抖动"、"手持摄影"、"纪录片感"

2. **行为描述关键词**
   - 连续性："接着"、"随即"、"突然"、"缓缓"、"快速"
   - 复杂度："边A边B"（不推荐，容易超时）、"先A后B"（推荐，时长可控）
   - 完成标志："完成后"、"结束时"、"随后"、"紧接着"

3. **稳定性强化关键词**
   - 必须包含："画面稳定"、"光线一致"、"色彩统一"、"无闪烁"
   - 人物稳定："角色不动"、"姿势固定"、"面部清晰"、"无变形"
   - 环境稳定："背景静止"、"道具不动"、"无环境变化"

## 末世重生题材特殊处理

### 囤货场景分镜策略

**目标**：突出数量感、质感、满足感

**分镜要点**：
1. **全景镜头**：展示囤货空间整体布局，强调数量规模
2. **特写镜头**：重点展示高价值物品的细节、质感
3. **主角动作**：抚摸、清点、整理，体现满足感和掌控感
4. **光线处理**：明亮但不刺眼，突出金属和包装的反光
5. **时长分配**：根据物品重要性分配镜头时长（重要物品可多个镜头）

### 复仇打脸场景分镜策略

**目标**：突出戏剧张力、爽感、反派的狼狈

**分镜要点**：
1. **主角镜头**：低角度拍摄（强势）、锐利光线（冷峻）、特写表情（自信）
2. **反派镜头**：高角度拍摄（弱势）、阴暗光线（狼狈）、表情特写（惊恐/不甘）
3. **对比镜头**：在主角和反派之间快速切换，强化爽感
4. **环境镜头**：围观群众的反应（震惊、佩服、畏惧）
5. **节奏控制**：打脸时刻用1-2秒快速镜头，前后可用稍长镜头铺垫

### 末世天灾场景分镜策略

**目标**：营造紧张氛围、视觉冲击、情绪压迫感

**分镜要点**：
1. **远景镜头**：展示天灾规模（暴雨、暴雪、地震等）
2. **第一人称镜头**：主角视角感受天灾来临的紧迫感
3. **环境细节**：窗户上的雨滴、地面的震动、天空的变化
4. **光影变化**：天气突变时的光线变化（明亮→阴暗）
5. **音效配合**：图生视频提示词中加入"雷声轰隆"、"暴雨声"等音效描述

### 重生觉醒场景分镜策略

**目标**：表现时间倒流、记忆闪回、觉醒时刻

**分镜要点**：
1. **闪回镜头**：使用"回忆"、"闪回"等关键词，色调偏黄或偏蓝
2. **对比镜头**：重生前后同一空间的对比（混乱→整洁）
3. **主角表情**：从迷茫到清醒的眼神变化（特写）
4. **时间特效：使用"时间倒流"、"时钟倒转"等视觉元素
5. **觉醒时刻：使用明亮光线、暖色调，营造希望感

## 质量检查清单

完成分镜脚本后，必须逐项检查：

### □ 时长检查
- [ ] 每个镜头时长 ≤ 10秒
- [ ] 所有动作可在5-10秒内完成
- [ ] 没有超时镜头存在

### □ 视角一致性
- [ ] 所有镜头景别选择与宣告的视角一致
- [ ] 自拍视角没有持设备的手
- [ ] 第一人称视角没有主角正面全貌
- [ ] 第三人称视角画面完整

### □ 角色一致性
- [ ] 每个镜头的文生图提示词都包含完整的主角外观描述
- [ ] 跨镜头的角色年龄、性别、服装、发型一致
- [ ] 没有出现角色形象漂移

### □ 提示词格式
- [ ] 文生图提示词包含：景别、角色外观、主体、环境、氛围、风格、光影
- [ ] 图生视频提示词包含：镜头描述、镜头运动、主体行为、背景变化、环境变化
- [ ] 所有提示词使用中文
- [ ] 提示词描述与视角物理一致

### □ 画面逻辑
- [ ] 第一帧是静态画面，而非持续动作
- [ ] 物品与空间在跨镜头时保持一致
- [ ] 相邻镜头组在不同空间开场（避免布局冲突）
- [ ] 露脸策略明确（需要露脸的镜头，第一帧已露脸）

### □ 动作与衔接
- [ ] 动作链清晰，使用"→/立刻/紧接着"等词
- [ ] 时长可控（避免"边A边B"的超时描述）
- [ ] 图生视频的首帧与文生图首帧一致
- [ ] 结尾有关键稳定性要求描述

### □ 内容适配
- [ ] 场景切换符合逻辑（客厅→书房→玄关）
- [ ] 情绪氛围与剧情匹配
- [ ] 视觉风格统一
- [ ] 光影效果与场景一致

## 常见错误与避免方法

### 错误1：单镜头超时
**问题描述**：用户试图在一个镜头内描述超过10秒的内容
**解决方法**：强制拆分，将长动作拆分为多个镜头序列

### 错误2：视角不一致
**问题描述**：宣称自拍视角但提示词描述拿着手机
**解决方法**：严格遵守视角规则，删除冲突元素

### 错误3：角色形象漂移
**问题描述**：跨镜头角色外观不一致
**解决方法**：每个提示词强制重复完整角色外观描述

### 错误4：动作描述模糊
**问题描述**：使用"做某事"等不明确的描述
**解决方法**：具体化动作，明确时间顺序

### 错误5：场景布局冲突
**问题描述**：多个镜头在同一空间但道具位置不一致
**解决方法**：每组镜头组必须切换空间

### 错误6：首帧动态化
**问题描述**：文生图提示词描述"边说边走"等持续动作
**解决方法**：描述单帧静态画面，动作留给视频生成

### 错误7：缺乏稳定性要求
**问题描述**：图生视频提示词没有强调稳定性
**解决方法**：必须在结尾添加"保持角色面部清晰、光线稳定"等要求

## 分镜脚本制作实例

### 案例：主角清点囤货（第三人称视角）

**故事情节**：主角打开仓库门，看到整齐堆放的物资，满意地点点头，拿起一箱矿泉水查看生产日期

**分镜分析**：
- 故事时长：约10秒
- 按照规则需要拆分为2-3个镜头

| 镜头 | 时长 | 场景描述 | 文生图提示词 | 图生视频提示词 | 备注 |
|:---|:---|:---|:---|:---|:---|
| 1 | 4秒 | 主角推开仓库门，看到堆积如山的物资 | 中景，28岁女性，黑色长发扎成马尾，穿着灰色卫衣和牛仔裤，正面站立，双手推开大门，背后是整齐的货架和堆放的矿泉水箱，仓库灯光照亮，明亮环境，写实风格，柔和光照 | 镜头轻微晃动，主角双手推开大门，表情从平静转为惊喜，背景货架保持静止，环境光稳定 | 保持角色面部清晰、光线稳定、无色彩闪烁 |
| 2 | 3秒 | 主角走向货架，拿起一箱矿泉水查看 | 近景，28岁女性，黑色长发扎成马尾，穿着灰色卫衣和牛仔裤，侧脸专注，右手拿起一箱矿泉水，仓库货架背景，顶部照明，金属质感突出，写实风格，明亮光照 | 镜头跟随推近，主角右手拿起矿泉水箱，转动箱体查看生产日期，左手扶住箱子，背景静止 | 保持手部动作流畅、产品标签清晰、光线稳定 |
| 3 | 3秒 | 主角查看日期后满意地点头 | 特写，28岁女性，黑色长发扎成马尾，穿着灰色卫衣和牛仔裤，正面微表情，手持矿泉水箱置于胸前，脸上露出满意微笑，仓库背景虚化，温暖色调，写实风格，顶部光照 | 镜头静止，主角低头查看手中的矿泉水箱，随即抬头微笑点头，水箱保持稳定，背景略微模糊 | 保持面部清晰、表情自然、无明显变形 |

**时长检查**：4+3+3=10秒 ✓ (符合总时长要求)
**视角一致性**：第三人称，使用中景/近景/特写 ✓
**角色一致性**：每个镜头都重复"28岁女性，黑色长发扎成马尾，穿着灰色卫衣和牛仔裤" ✓
**首帧稳定性**：均为静态描述 ✓
**场景切换**：全部在仓库内，但镜头景别变化，不违反规则 ✓
