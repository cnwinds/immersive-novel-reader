# 职场逆袭单集正文模板

## 基本信息
- **集数编号**：Episode-XX
- **标题**：[5-8字概括核心情节]
- **字数要求**：500-800字
- **结构要求**：起承转钩四段式

---

## 正文模板

### 起（100-150字）
**功能**：快速建立职场冲突，展现主角处境

**标准结构**：
```
[时间/场景交代] (1句)
[职场环境描述] (2-3句)
[核心冲突引入] (3-5句)
```

**示例**：
```
周一上午，星云科技会议室里。

气氛紧张得让人窒息。

产品总监张建国冷笑着看她。

"就这点能力？"

他把文件扔在桌上。

所有人都看向她。
```

---

### 承（200-300字）
**功能**：冲突升级，主角应对，推进剧情

**标准结构**：
```
[主角应对策略] (5-8句)
[职场对话交锋] (5-8句)
[矛盾激化过程] (5-8句)
```

**示例**：
```
林小雨深吸一口气，站了起来。

她打开笔记本电脑，连接投影仪。

PPT开始在大屏幕上播放。

"各位领导，请看我的方案。"

数据分析详实，逻辑清晰。

创意让在场所有人眼前一亮。

张建国的表情开始变化。

客户频频点头。

这个方案超出预期。
```

---

### 转（150-200字）
**功能**：爽点高潮，主角能力展示或反击成功

**标准结构**：
```
[关键行动] (3-5句)
[能力展示] (5-8句)
[对手反应] (3-5句)
```

**示例**：
```
"就用这个方案！"

客户当场拍板决定。

张建国脸色惨白，说不出话。

所有人都看向林小雨。

掌声响了起来。

她成功地证明了。

自己的能力。

今天的胜利来之不易。

```

---

### 钩（50-100字）
**功能**：设置悬念，引出下集

**标准结构**：
```
[新挑战出现] (1-2句)
[关系转折] (2-3句)
[留下悬念] (1-2句)
```

**示例**：
```

会后，李明辉走了过来。

"林小雨，你很厉害。"

他的眼中带着异样的光芒。

"我们聊聊？"

```

## 写作规范

### 句式要求
- **平均句长**：8-15字
- **最长不超过**：20字
- **短句为主**：营造快节奏感
- **避免长句**：超过25字必须拆分

### 内容要求
- **职场真实性**：环境、对话、行为要符合现代职场
- **爽点明确**：每集至少1-2个打脸或能力展示点
- **节奏紧凑**：避免拖沓，快速推进剧情
- **悬念设置**：每集结尾都要为下集铺垫

### 对话要求
- **身份相符**：符合角色的职场身份和专业背景
- **信息量大**：每句对话都要包含有效信息
- **冲突驱动**：通过对话推进矛盾和冲突
- **简洁有力**：避免冗长和废话

### 描写要求
- **视觉化强**：每段都要能转化为具体画面
- **动作具体**：用具体动词表现人物行为
- **环境烘托**：用职场环境营造氛围
- **心理简洁**：心理描写控制在1-2句内

---

## 爽点类型

### 能力打脸爽点
**表现形式**：专业技能展示，解决重大问题
**创作要点**：
- 前期铺垫：主角被质疑、被低估
- 能力展示：关键时刻展现专业技能
- 结果震撼：超出预期，获得认可

### 晋升爽点
**表现形式**：获得晋升机会，职位提升
**创作要点**：
- 业绩支撑：有明确的工作成果
- 能力证明：展现岗位所需素质
- 情感升华：获得认可和尊重

### 复仇爽点
**表现形式**：揭露反派阴谋，正义获得伸张
**创作要点**：
- 前期委屈：主角遭受不公待遇
- 能力成长：今时不同往日
- 正义伸张：通过规则让反派自食恶果

## 创作提示

### 环境描写技巧
- 抓住1-2个核心视觉元素
- 用环境烘托人物心理
- 体现职场特色和氛围

### 对话创作技巧
- 每句话都要推动剧情
- 体现人物性格和身份
- 创造冲突和张力

### 节奏控制技巧
- 短句创造紧张感
- 长句舒缓情绪（少量使用）
- 用段落分割控制节奏

---

## 备注

- 模板提供的是标准格式，可根据具体剧情调整
- 所有创作都必须基于现代职场现实
- 爽点的呈现要自然，不要过于刻意
- 每集都要有明确的价值体现和观众共鸣点
