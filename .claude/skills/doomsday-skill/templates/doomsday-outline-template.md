# 故事大纲模板

## 基本信息
- **故事标题**：[8-12字，体现末世+重生+核心冲突]
- **创作日期**：[YYYY-MM-DD]
- **总集数**：60集
- **单集字数**：500-800字

## 故事梗概（100-150字）
[用一段话概括整个故事，必须包含：主角身份+前世死因+重生优势+核心冲突+终极目标]

示例格式：
前世，[主角姓名]被[前世仇人]害死在[地点/情境]。重生到天灾爆发前[X个月]，她获得了[金手指类型]，开始疯狂囤货准备。这一世，她要[复仇目标]，同时在[天灾类型]中带领[团队/家人]活下去。从[初始状态]到[最终状态]，她将用前世的教训和今世的力量，改写命运。

## 三幕式结构规划

### 第一幕：重生起点与初期囤货（第1-20集）

**核心任务**：
- 建立重生优势
- 完成基础囤货
- 小规模复仇

**关键情节点**：
- 第1-2集：[前世死亡回忆，展示如何被害、谁是凶手、主角重生时间节点]
- 第3-5集：[金手指获得经过、初步测试能力、确定能力边界]
- 第6-15集：[囤货计划，包括食物、水、药品、武器等具体类别和数量]
- 第10-18集：[炮灰反派登场和打脸节奏，列出3-5个小反派及打脸节点]
- 第15-20集：[天灾前兆显现，社会开始混乱，主角预警家人/朋友]

**本幕爽点分布**：
- 囤货爽：[具体列出5-8个囤货展示场景]
- 打脸爽：[具体列出3-5个炮灰反派打脸场景]
- 预知爽：[具体列出2-3个提前预知危险的场景]

**本幕结束状态**：
- 主角已囤积：[具体物资清单]
- 已打脸：[已处理的炮灰反派名单]
- 天灾状态：[即将爆发/已有前兆]

### 第二幕：天灾升级与中期扩张（第21-45集）

**核心任务**：
- 天灾正式爆发
- 基地建立与扩张
- 核心冲突展开

**关键情节点**：
- 第21-25集：[天灾爆发具体情节，社会秩序崩溃，普通人反应]
- 第26-32集：[基地建立过程，选址、防御、资源配置]
- 第30-38集：[幸存者聚集，盟友加入，敌人识别]
- 第33-40集：[前世仇人登场，揭露前世恩怨，今世初步冲突]
- 第35-45集：[资源争夺战，与其他势力竞争]
- 第38-45集：[金手指升级，解锁新功能，能力成长]
- 第40-45集：[感情线推进，主角与伴侣/队友情感深化]

**本幕爽点分布**：
- 囤货回报：[具体列出5-8个囤货优势展示场景]
- 能力碾压：[具体列出5-8个金手指优势场景]
- 复仇推进：[具体列出3-5个接近前世仇人的布局场景]
- 团队凝聚：[具体列出2-3个收服人心的场景]

**本幕结束状态**：
- 基地规模：[多少人，什么设施]
- 核心团队：[列出主要成员名单]
- 与前世仇人关系：[已发现身份/已初步交锋/已布局陷阱]
- 金手指状态：[当前等级和能力]

### 第三幕：终极对决与收官（第46-60集）

**核心任务**：
- 复仇高潮
- 终极危机解决
- 故事收尾

**关键情节点**：
- 第46-52集：[前世仇人对决，揭露前世全部真相，复仇过程]
- 第50-56集：[终极危机爆发，最高等级天灾或隐藏boss出现]
- 第52-58集：[金手指达到巅峰，主角最强状态]
- 第54-58集：[势力决战，主角阵营vs敌对势力]
- 第55-58集：[复仇完成，前世仇人结局]
- 第58-60集：[新秩序建立，主角地位确立，展望未来]

**本幕爽点分布**：
- 复仇高潮：[具体列出前世仇人的惩罚场景2-3个]
- 力量爆发：[具体列出金手指全开的震撼场景2-3个]
- 情感升华：[具体列出与伴侣/队友的生死时刻1-2个]
- 终极胜利：[具体描述最终胜利的画面]

**本幕结束状态**：
- 前世仇人结局：[具体下场]
- 主角最终地位：[在新世界的地位和影响力]
- 团队状态：[核心成员存活情况]
- 结局基调：[开放式/圆满/留悬念]
