# status - 查看创作进度

**命令说明**：检查项目文件夹，显示当前小说创作的进度情况，帮助用户了解完成了哪些部分，还有哪些待完成。

**使用格式**：
```
/status
```

**使用场景**：随时查看创作进度，或在长时间后重新开始创作时了解当前状态。

## 功能说明

执行本命令后，系统将：

1. **检查核心文档状态**
   - [x] 检查 `outline.md` 是否存在（故事大纲）
   - [x] 检查 `characters.md` 是否存在（人物小传）
   - [x] 检查 `catalog.md` 是否存在（章节目录）

2. **检查章节文件状态**
   - [x] 检查 `episodes/Episode-01.md` 是否存在（第1章）
   - [x] 检查 `episodes/Episode-02.md` 是否存在（第2章）
   - [x] 检查 `episodes/Episode-03.md` 是否存在（第3章）
   - [x] 检查 `episodes/Episode-04.md` 是否存在（第4章）
   - [x] 检查 `episodes/Episode-05.md` 是否存在（第5章）

3. **显示创作进度**

   示例输出：
   ```
   现代言情小说创作进度：

   === 核心文档 ===
   [✓] 故事大纲 (outline.md) - 已创建
   [✓] 人物小传 (characters.md) - 已创建
   [✓] 章节目录 (catalog.md) - 已创建

   === 章节进度 ===
   [✓] 第1章：相遇与初印象 - 已完成 (2,500字)
   [✓] 第2章：关系深化 - 已完成 (2,200字)
   [✓] 第3章：情感升温 - 已完成 (2,800字)
   [✓] 第4章：分离与反思 - 已完成 (2,400字)
   [ ] 第5章：和解与圆满 - 待写作

   === 整体进度 ===
   已完成：80% (4/5章)
   预计还需：1-2小时

   === 下一步建议 ===
   使用命令：/write 5
   开始创作第5章：和解与圆满
   ```

```
[ ] 章节目录 (catalog.md) - 待创建

建议下一步：
使用命令：/outline
开始创建你的故事大纲
```
