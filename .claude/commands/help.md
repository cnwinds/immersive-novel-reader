# help

**描述**: 显示完整指令列表和使用说明

**用法**: `/help [命令名]`

**参数**: 
- `命令名` (可选): 查看特定命令的详细帮助

---

## 执行流程

当用户调用 `/help` 命令时:

1. **判断调用方式**
   - 如果用户只输入 `/help`,显示完整命令列表和快速入门
   - 如果用户输入 `/help <命令名>`,显示该命令的详细说明

2. **显示完整帮助信息** (无参数时)

   ```

   ═══════════════════════════════════════════

   创作流程命令:
   /outline    - 创建或修改故事大纲

   /catalog    - 生成或修改60集分集目录
   /write      - 开始或继续正文创作
   
   辅助工具命令:
   /check      - 手动触发质量检查
   /status     - 显示当前创作进度
   /help       - 显示本帮助信息

   🚀 快速入门
   
   第一次使用? 按照以下步骤开始:
   
   步骤1️⃣: 创建故事大纲
   输入: /outline
   系统会引导你完成Q1-Q3创意问答,然后生成完整大纲
   
   步骤2️⃣: 创建人物小传
   输入: /character
   基于大纲自动生成主角、反派、配角等人物设定
   
   步骤3️⃣: 生成分集目录
   输入: /catalog
   自动生成60集完整目录,包含三幕式结构布局
   
   步骤4️⃣: 开始正文创作
   输入: /write
   系统会自动创作5集内容,完成质量检查后写入文件
   继续输入 /write 可以持续创作下一批
   
   ═══════════════════════════════════════════
   
   💡 使用技巧
   
   - 随时输入 /status 查看创作进度

   - 如果某集内容不满意,可以删除后重新 /write
   - 正文创作必须按顺序,不能跳集
   - 大纲/人物/目录可以随时修改
   
   ═══════════════════════════════════════════
   
   ❓ 查看详细帮助
   
   输入 /help <命令名> 查看特定命令的详细说明
   例如: /help outline

   ✨ 质量保障: 每批5集自动检查,确保不偏离大纲
   ```

3. **显示特定命令帮助** (有参数时)

   根据用户指定的命令名,显示该命令的详细信息:
   
   - 命令描述
   - 使用方法
   - 参数说明
   - 前置条件
   - 执行流程概述
   - 使用示例
   - 注意事项

   例如用户输入 `/help outline`:
   
   ```
   📖 命令详解: /outline
   
   ═══════════════════════════════════════════
   
   📝 描述
   创建或修改故事大纲(会询问是否覆盖现有大纲)
   
   💻 用法
   /outline
   
   📋 参数
   无参数
   
   ⚠️  前置条件
   无(这是创作流程的第一步)
   
   ═══════════════════════════════════════════
   
   🔄 执行流程
   
   1. 检查是否已有大纲文件
   2. 收集用户创意(Q1-Q3)
   3. 读取大纲创作方法论和模板
   4. 生成完整大纲(包含故事梗概、三幕结构、
      天灾体系、金手指、核心冲突、爽点规划)
   5. 写入 outline.md 文件
   6. 提示下一步操作
   
   ═══════════════════════════════════════════

   用户: /outline
   系统: 开始收集创意...
         Q1: 请描述天灾类型和核心设定

   ═══════════════════════════════════════════

   - 固定60集结构,不支持自定义集数
   - Q1-Q3的回答质量直接影响大纲质量
   - 大纲完成后可随时修改
   - 后续所有创作都基于这个大纲
   
   ═══════════════════════════════════════════
   
   输入 /help 返回命令列表
   ```

4. **处理无效命令名**

   如果用户输入了不存在的命令名,显示:
   
   ```
   ❌ 未找到命令: /xxx
   
   可用命令列表:
   - /outline
   - /character
   - /catalog
   - /write
   - /check
   - /status
   - /help
   
   输入 /help 查看完整帮助
   ```

## 支持的命令参数

- `/help` 或 `/help ` - 显示完整帮助
- `/help outline` - outline命令详细说明
- `/help character` - character命令详细说明
- `/help catalog` - catalog命令详细说明
- `/help write` - write命令详细说明
- `/help check` - check命令详细说明
- `/help status` - status命令详细说明

## 注意事项

- **任何时候都可调用**: 无需前置条件
- **智能识别**: 自动识别带 / 或不带 / 的命令名
- **友好提示**: 输入错误命令会给出正确的命令列表
- **快速入门**: 首次使用建议直接查看完整帮助
