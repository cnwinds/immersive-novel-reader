## 本示例说明

### 分镜概述
- **总时长**：5+5+4+5+6+5 = **30秒** ✓
- **镜头数量**：6个镜头
- **平均每镜时长**：5秒（符合5-10秒范围）
- **视角**：第三人称视角
- **风格**：写实风格
- **场景**：别墅地下室仓库（同一空间）

### 分镜头设计思路

**镜头1：地下室的入口**
- **景别**：中景（交代人物与环境关系）
- **重点**：开锁动作、钥匙串细节、门口环境
- **情绪**：紧张而期待（囤货成果即将呈现）
- **时长**：5秒（合理的开锁动作时间）

**镜头2：适应黑暗**
- **景别**：近景（推进观众与角色的距离）
- **重点**：开门的物理动作、适应黑暗的过程
- **情绪**：警惕到期待
- **时长**：5秒（符合开灯、适应光线的真实节奏）

**镜头3：灯光亮起**
- **景别**：近景（聚焦于主角的脸部反应）
- **重点**：强光的视觉冲击、主角表情变化、物资轮廓
- **情绪**：震撼、满意、成就感
- **时长**：4秒（快速的适应过程）

**镜头4：查看细节**
- **景别**：特写（突出物品的质感和细节）
- **重点**：矿泉水箱标签、生产日期、主角的谨慎态度
- **情绪**：专业的囤货者、求生意志的体现
- **时长**：5秒（仔细查看商品的时间）

**镜头5：情感的流露**
- **景别**：中景（展示人物与环境的关系）
- **重点**：蹲姿、抚摸罐头、眼神中的安全感
- **情绪**：温柔、珍惜、对未来的掌控感（爽点体现）
- **时长**：6秒（情感的酝酿与释放）

**镜头6：收束镜头**
- **景别**：中景（综合展示人物与环境）
- **重点**：拥抱的动作（与物资的情感连接）、锁门（安全意识）
- **情绪**：满足、坚定、自豪感
- **时长**：5秒（完整的收尾动作）

### 空间一致性设计

**同一空间内的物品位置保持一致**：

- **金属门**：所有镜头中的门都在同一位置（左侧）
- **货架/物资**：镜头3-6中，货架始终位于背景右侧
- **光线**：所有镜头都保持顶部照明的一致性
- **避免冲突**：虽然全部镜头在同一地下室，但镜头3-6的背景布局保持一致，防止AI生图幻觉

### 爽点设置

本场景集中体现了末世重生漫剧的**囤货爽点**：

| 爽点类型 | 镜头体现 | 效果 |
|:---|:---|:---|
| **物资充沛** | 镜头3-6 | 货架堆满物资堆到天花板，视觉冲击强烈 |
| **掌控感** | 镜头4、5 | 主角查看日期、抚摸罐头，体现对物资的完全掌控 |
| **安全感** | 镜头5、6 | 拥抱动作、锁门动作，展示内心的安全感 |
| **重生优势** | 全场景 | 与前世的物资匮乏形成对比，体现重生价值 |

### 图生视频提示词的特点分析

每个镜头的图生视频提示词都有以下特点：

1. **动作链清晰**：用"随即"、"然后"、"随后"串联动作
2. **时长可控**：描述的都是5-10秒内可完成的动作
3. **结尾稳定性要求**：每个提示词结尾都有"保持角色面部清晰..."
4. **首帧衔接**：所有视频的起始帧都是对应文生图的静态画面

## 借鉴本示例的要点

### 可以学习的方面

1. **角色描述完整度**：如何详细描述角色外貌并复制到每个分镜
2. **场景布局逻辑**：同一空间内的物品位置如何保持一致
3. **情绪递进设计**：从期待→震撼→满意→安全的完整情绪线
4. **爽点分布**：在6个镜头中均匀分布囤货爽感
5. **时长合理性**：每个镜头的动作都在5-10秒内合理完成
6. **提示词工程**：文生图和图生视频提示词的格式和技巧

### 不建议模仿的方面

1. **空间使用率**：本示例6个镜头均在同一场景（地下室），实际创作中建议每组镜头切换不同空间，以避免布局冲突，并增加视觉丰富性。

2. **建议改进**：如果重新设计，可将"查看物资"（镜头4）移至"客厅"或"书房"进行，实现"地下室→客厅"的场景切换，更符合方法论要求。

## 错误示范与对比

### 错误示例（不使用本模板的情况）

**❌ 文生图提示词过于简单**：
```
错误示例：一个女人在地下室看物资
问题分析：
- 没有年龄、外貌描述（角色不一致）
- 没有场景细节（画面空洞）
- 没有情绪氛围（平铺直叙）
- 没有光照描述（画面平淡）
- 不符合格式规范
```

**❌ 图生视频提示词超时**：
```
错误示例：主角走进仓库，查看所有物资，然后锁门离开
问题分析：
- "查看所有物资"这个动作可能超过30秒，违反5-10秒规则
- 多个动作合并描述（应拆分为多个镜头）
- 首帧不是静态画面（"走进"是持续动作）
```

**❌ 视角描述冲突**：
```
错误示例（宣称自拍视角）：自拍角度，主角手持手机自拍杆
问题分析：
- "手持自拍杆"破坏了自拍视角的沉浸感
- 自拍视角不应看到持设备的手
```

## 注意：示例的局限性

本示例为了完整展示一个场景的所有镜头，**全部6个镜头都设置在同一个地下室空间**内，这在实际创作中可能因AI生图幻觉导致跨镜头的场景布局不一致。

**建议实际创作中**：
- 每组3-4个镜头后更换场景
- 例如：地下室囤货 → 客厅清点 → 书房记录 → 卧室休息
- 这样可以确保每个场景的道具布局稳定
